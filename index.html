Installation Instructions

Download HENS from GitHub.

HENS (HTML Educational Node.js System) is an applied system for web development. It provides an HTML & Node.js development environment that runs off a USB drive or cloud storage.

Using HENS or your Node.js environment, run the following command:

node app.js
